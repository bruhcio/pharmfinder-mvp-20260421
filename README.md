# PharmFinder MVP

약국 재고 정보 전달 중심의 실제 실행 가능한 React 웹사이트 MVP입니다.

## 포함 내용

- 소비자 약품 검색 UI
- 사진 업로드 기반 검색 진입 데모
- 내 위치 기반 가까운 약국 추천
- 재고 많이 보유한 약국 추천
- 약국별 재고 결과 카드
- 슬라이드형 약국 상세 패널
- 약사 재고 수정 화면
- 반응형 레이아웃
- 부드러운 hover / press 인터랙션
- 전화 문의 / 길찾기 정보성 액션
- 정보 제공 중심 법률 리스크 완화 문구
- `Vite + React` 기반 실제 웹앱 구조

## 실행 방법

```bash
npm install
npm run dev
```

기본 개발 서버 주소는 `http://localhost:4173` 입니다.

배포 빌드는 아래 명령으로 생성합니다.

```bash
npm run build
```

## 현재 포지션

- 이 버전은 의약품 판매/주문/예약 서비스가 아니라 `약국 재고 정보 전달` 중심 MVP입니다.
- 사용자에게는 재고 참고 정보, 약국 연락처, 길찾기, 위치 기반 추천을 제공합니다.
- 실제 구매 가능 여부와 복약 상담은 약국 약사 확인을 전제로 합니다.

## 실제 서비스 연결 시 필요한 것

- 네이버 지도 API 키
- 공공데이터포털 약국/의약품 API 키
- 약국 POS/ERP 연동용 제휴 또는 커넥터
- OCR 또는 비전 기반 사진 인식 API
- 백엔드 및 데이터베이스

## 배포

- GitHub Pages 배포 워크플로우를 포함했습니다.
- 저장소의 Pages 설정이 GitHub Actions 배포를 사용하도록 켜져 있으면 자동 배포할 수 있습니다.
